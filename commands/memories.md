# Memories

List all memories for the current agent and project.

Options:
- `--kind TYPE` - Filter by type (EMOTIONAL, ARCHITECTURAL, LEARNINGS, ACHIEVEMENTS)
- `--region REGION` - Filter by region (AGENT, PROJECT)
- `--all` - Include superseded memories

$ARGUMENTS

```bash
uv run python -m ltm.commands.memories $ARGUMENTS
```
